## Etapa 1 : Fundamentos de Git

# Descripcion Etapa

Este repositorio fue creado para aprender y aplicar los fundamentos de Git y la interconexion a github, aplicando buenas practicas de control de versiones usadas en entornos profesionales.

## Objetivo del Proyecto

-Comprender el flujo basico de Git (add,commit y push)
-Usar ramas para generar entornos de trabajo seguros
-Gestionar un repositorio remoto en GitHub
-Mantener un historial de cambios claro y ordenado

## Tecnologias Utilizadas

-Git
-GitHub

## Contenidos del repositorio

-readme.md : Documentacion del proyecto
-Ejercicios practicos de Git
